# P.Q.P. — Pra Quem Pode (Hotseat)

Versão **passa o celular** (hotseat), 100% estática, sem servidor.

## Como usar (GitHub Pages)
1) Crie um repositório no GitHub
2) Envie estes arquivos para a **raiz** do repo
3) Settings → Pages → Deploy from branch → `main` / `/root`
4) Abra o link do Pages no celular

## Instalar como app (PWA)
- iPhone: Safari → Compartilhar → **Adicionar à Tela de Início**
- Android: Chrome → menu → **Instalar app**

## Regras rápidas
- 1 anfitrião por rodada (julga, não responde)
- anfitrião escreve a frase com `...` (lacuna)
- cada jogador, na sua vez, escreve resposta
- anfitrião escolhe a melhor, revela quem foi, +1 ponto
- quem ganhou vira o novo anfitrião
- vence quem atingir a meta de pontos
